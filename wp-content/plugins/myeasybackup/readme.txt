== myEASYbackup Installation ==
This section describes how to install the plugin and get it working.

1. Upload the full directory into your `wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in the WordPress Administration page
3. Open the plugin settings page, which is located under `Settings -> myEASYbackup`
4. Set the options as you like
5. Open the plugin tools page, which is located under `Tools -> myEASYbackup`
6. Click on the backup button and wait myEASYbackup to prepare the backup data set
7. When the backup data set is ready click on the save button to get it on your system
